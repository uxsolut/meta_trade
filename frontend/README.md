# Gestor de Capitais

**IA de Gestão de Ativos** - Controle e transforme seus investimentos com inteligência artificial avançada.

## 🚀 Sobre o Projeto

O Gestor de Capitais é uma plataforma web/mobile desenvolvida em Flutter que oferece uma solução completa para gestão profissional de ativos através de inteligência artificial avançada.

### ✨ Características Principais

- **Design Mobile-First**: Otimizado para dispositivos móveis e web
- **Animações Fluidas**: Efeitos visuais modernos e responsivos
- **Menu Interativo**: Menu dropdown animado com efeitos hover
- **Interface Elegante**: Design minimalista com gradientes sofisticados
- **Navegação Preparada**: Estrutura pronta para múltiplas páginas

## 🛠️ Tecnologias Utilizadas

- **Flutter 3.0+**
- **Dart**
- **Material Design**
- **Custom Animations**
- **Responsive Design**

## 📱 Funcionalidades

### Seções Implementadas

1. **Hero Section** - Apresentação principal com animações

## 🚀 Como Executar

### Pré-requisitos

- Flutter SDK 3.0 ou superior
- Dart SDK
- Android Studio / VS Code
- Emulador Android/iOS ou dispositivo físico

### Instalação

1. **Clone o projeto**
   ```bash
   git clone <url-do-repositorio>
   cd gestor_capitais
   ```

2. **Instale as dependências**
   ```bash
   flutter pub get
   ```

3. **Execute o projeto**
   ```bash
   flutter run
   ```

### Para Web

```bash
flutter run -d chrome
```

### Para Build de Produção

```bash
# Android
flutter build apk --release

# iOS
flutter build ios --release

# Web
flutter build web --release
```
